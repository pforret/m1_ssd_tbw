# m1_ssd_tbw

Measure SSD wear (TBW) on MacOS M1 (ARM64) machines

## 🚀 Installation

with [basher](https://github.com/basherpm/basher)

	$ basher install pforret/m1_ssd_tbw

or with `git`

	$ git clone https://github.com/pforret/m1_ssd_tbw.git
	$ cd m1_ssd_tbw

## 🔥 Usage

	> m1_ssd_tbw.sh [action]
